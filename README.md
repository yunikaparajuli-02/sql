# Foundation of Computer Science – Coursework Repository  
**Module Code:** ST4015CMD  
**Module Name:** Foundation of Computer Science  

This repository contains all **practical implementations, scripts, and supporting files** for the coursework of the *Foundation of Computer Science* module at **Softwarica College of IT & E-Commerce**.  
The work in this repository supports the written report and demonstrates key theoretical and practical concepts covered in the module.

---

##  Overview of Tasks

This repository includes work for **Task 1, Task 2, and Task 3**, focusing on encoding, logic, problem solving, and data management.

##  Task 1: Encoding and Decoding (Security & Data Transmission)

This task demonstrates the use of **encoding formats** to represent and transmit data securely.

### Concepts Covered:
- ASCII encoding
- Base64 encoding and decoding
- Role of encoding in secure data exchange

### Implementation:
- Converts a text message into ASCII values
- Encodes the message using Base64
- Decodes the Base64 message back to the original text

**Technology Used:** Python  

##  Task 2: Boolean Logic and Problem Solving

This task applies **Boolean logic** and **computational thinking** to solve logical and constraint-based problems.

### Concepts Covered:
- Boolean variables
- Logical operators (AND, OR)
- Satisfiability checking
- Understanding P vs NP through real-world scenarios

### Implementation:
- Logical conditions modeled using Boolean expressions
- Constraint checking using Z3 Solver
- Demonstrates how rules can be verified efficiently

**Technology Used:** Python, Z3 SMT Solver  



## Task 3: Data Modeling and Database Normalization

This task focuses on **data organization and management** using relational database concepts.

### Concepts Covered:
- Data redundancy and anomalies
- First, Second, and Third Normal Form (1NF, 2NF, 3NF)
- Entity Relationship (ER) modeling
- SQL operations and JOIN queries

### Implementation:
- Separation of data into Student, Club, and Membership tables
- Use of primary and foreign keys
- SQL queries for inserting, retrieving, and joining data

**Technology Used:** SQL, Xampp
